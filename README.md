# ModelEngineBridge

Provides a minimal bridge to allow limited API support for Model Engine v3 and v4.

## API

Repository:
```kts
repositories {
    maven("https://repo.auxilor.io/repository/maven-public/")
}
```

Dependency:
```kts
dependencies {
    implementation("com.willfp:ModelEngineBridge:1.0.0")
}
```
